# 6.1.1. Latihan 1
- Buatlah sebuah class yang bersi variabel
  - nama, 
  - nim dan 
  - alamat 
  - serta sebuah *constructor default* yang dapat menampilkan 
     - nama, 
     - nim dan 
     - alamat Saudara. 
- Selanjutnya buatlah kelas baru dengan fungsi ``main()`` di dalamnya untuk menampilkan hasilnya.
